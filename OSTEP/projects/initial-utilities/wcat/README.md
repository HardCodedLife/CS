## My Thoughts

### exit vs return
| Feature | exit | return |
| :--- | :--- | :--- |
| **Scope** | Return to the OS | Return to where function is called |
| **Impact** | Terminates the entire program | Leave the current function |
| **Best Practice** | Better be used at main | Could be used at any function |

### Edge case
- No arguments provided
- File not exist
- Multiple files are provided
